Pre-Lab Exercise
Pre-Lab Calculation
Before starting the experiment, calculate the expected [FeSCN²⁺] for each of the four standard solutions in Part I. In Part I, Fe³⁺ is in large excess, so ALL of the SCN⁻ will be converted to FeSCN²⁺.
Background — The Reaction
Fe³⁺(aq) + SCN⁻(aq) ⇌ FeSCN²⁺(aq)
FeSCN²⁺ forms a distinctive blood-red color in solution. The reactants (Fe³⁺ and SCN⁻) are essentially colorless, so all color observed is due to FeSCN²⁺.
The equilibrium expression is:
Keq = [FeSCN²⁺] / ([Fe³⁺][SCN⁻])
Pre-Lab: Calculate [FeSCN²⁺] for Part I Standard Solutions
In Part I, 5.0 mL of 0.200 M Fe³⁺ is added to each beaker. Since [Fe³⁺] ≫ [SCN⁻], essentially all SCN⁻ converts to FeSCN²⁺. Use dilution: [FeSCN²⁺] = (VSCN × MSCN) / Vtotal where Vtotal = 50.0 mL.
